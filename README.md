## 🚀 Getting Started

These instructions will help you set up the project locally for development.

### 📦 Prerequisites

Install frontend dependencies:

```bash
npm install
````

### ▶️ Running Locally

Start the frontend:

```bash
npm run dev
```

Start the backend (in `/server`):

```bash
node server.js
```

Visit [http://localhost:5173](http://localhost:5173) in your browser.
